# Changelog

## 1.1.1 - 2026-02-20

### Fixed

- 修复 smooth 客户端模式下服务端玩家位置不同步，导致区块加载与实体跟踪范围未跟随目标的问题。
- 修复目标玩家下线时客户端平滑镜头状态未正确重置的问题。

### Changed

- 精简 GitHub Release 正文模板，移除硬编码的版本列表与下载说明。

## 1.1.0 - 2026-02-11

### Added

- 新增客户端快捷键 `打开配置菜单`（默认 `F9`）。
- 新增移轴功能实验性风险提示与相关 i18n 文案。

## 1.0.9

### Added

- 新增客户端 mod 支持，改善旁观视角流畅度。

## 1.0.8

### Added

- 新增 `/cs stats` 与 `/cs top` 统计能力。
- 观察点支持分组；新增按组查看与按组加入循环。
- 新增 `/cs who` 查看全服旁观状态。
- Action Bar 状态展示增强。
- 旁观模式记忆能力。

### Changed

- 命令结构与模式参数统一，补充 `/cs` 简写链路。

## 1.0.7

### Added

- 新增 `/cs cycle addplayerall`，支持前后缀过滤。
- 自动处理玩家上下线与循环列表同步。

## 1.0.6

### Added

- 支持 Minecraft 1.21。

## 1.0.5

### Added

- 新增 `/cs config` 配置管理命令：`reload/get/set/list`。

### Changed

- 配置结构重构，增强可读性与可维护性。

## 1.0.4

### Added

- 新增电影视角系统与浮游镜头参数。

## 1.0.3

### Fixed

- 修复维度切换与玩家重进后的旁观状态恢复问题。

## 1.0.2

### Changed

- 配置与数据存储迁移为 JSON，并优化命令补全。

## 1.0.1

### Fixed

- 修复玩家重连后旁观状态恢复异常。

## 1.0.0

- 初始版本发布。
